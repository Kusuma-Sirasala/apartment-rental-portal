# Residential Apartment Rental Portal

## Overview
A full-stack mini project demonstrating a residential apartment rental system with
user registration, login, apartment browsing, and booking functionality.

## Tech Stack
- Frontend: HTML5, Tailwind CSS, JavaScript
- Backend: Python Flask (REST APIs)
- Database: Mock data (can be extended to PostgreSQL)
- Tools: VS Code, GitHub

## Features
### User Portal
- User Registration
- User Login
- Browse Apartments
- Book Apartment

### Backend APIs
- POST /api/register
- POST /api/login
- GET /api/apartments
- POST /api/book
